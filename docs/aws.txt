Connection to AWS
mac> ssh ubuntu@ec2-54-213-45-213.us-west-2.compute.amazonaws.com

Update All:
ubuntu 14.04> sudo apt-get update

Installing Mongo DB (https://docs.mongodb.org/manual/tutorial/install-mongodb-on-ubuntu/#install-mongodb-community-edition):
ubuntu 14.04> sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv EA312927
ubuntu 14.04> echo "deb http://repo.mongodb.org/apt/ubuntu precise/mongodb-org/3.2 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-3.2.list
ubuntu 14.04> sudo apt-get update
ubuntu 14.04> sudo apt-get install -y mongodb-org

Installing NodeJS (https://nodejs.org/en/download/package-manager/):
ubuntu 14.04> curl -sL https://deb.nodesource.com/setup_5.x | sudo -E bash -
ubuntu 14.04> sudo apt-get install -y nodejs

Installing Git:
ubuntu 14.04> sudo apt-get install -y git-core
ubuntu 14.04> git config --global user.name "aws-prod-user"
ubuntu 14.04> git config --global user.email "aws-prod-user@example.com"

Install Taskenize:

ubuntu 14.04> cd ~
ubuntu 14.04> git clone https://github.com/YuriyGorvitovskiy/Taskenize.git
ubuntu 14.04> cd ~/Taskenize/
ubuntu 14.04> mkdir OAuth

ubuntu 14.04> cd ~/Taskenize/WebClient/
ubuntu 14.04> mkdir br
ubuntu 14.04> npm install
ubuntu 14.04> ./node_modules/typings/dist/bin/typings.js install

ubuntu 14.04> cd ~/Taskenize/WebServer/
ubuntu 14.04> npm install
ubuntu 14.04> ./node_modules/typings/dist/bin/typings.js install

ubuntu 14.04> sudo cp scripts/taskenize.conf /etc/init.d/

Rebuild Taskenize>
mac> scp ~/Development/Taskenize/OAuth/config.aws.secret.js ubuntu@ec2-54-213-45-213.us-west-2.compute.amazonaws.com:~/Taskenize/OAuth/config.secret.js
ubuntu 14.04> cd ~/Taskenize/WebClient/
ubuntu 14.04> ./node_modules/.bin/tsc
ubuntu 14.04> ./bin/browserify.js

ubuntu 14.04> cd ~/Taskenize/WebServer/
ubuntu 14.04> ./node_modules/.bin/tsc


Mongo DB:
Start Mongo DB   > sudo service mongod start
Stop Mongo DB    > sudo service mongod stop
Restart Mongo DB > sudo service mongod restart
Log File         > cat /var/log/mongodb/mongod.log
